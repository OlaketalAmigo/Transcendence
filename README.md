# Transcendence

Exemple d'../.env fonctionnel:

POSTGRES_PASSWORD=coucou
POSTGRES_DB=database
POSTGRES_HOST=database
POSTGRES_USER=user


Gestion de friendship dans POSTGRESQL:
'pending' → demande envoyée
'accepted' → amis
'blocked' → bloqué
'rejected' → refusé

Ressource:
    https://www.postgresql.org/docs/
